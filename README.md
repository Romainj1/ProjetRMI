## Lancement du programme

Le programme nécessite une compilation. Le server et le client doivent être lancés sur des terminaux différents.

### Compiler le projet

```
./compile.sh
```

### Lancer le Serveur

```
./run.sh server
```

### Lancer le Client

```
./run.sh client
```
